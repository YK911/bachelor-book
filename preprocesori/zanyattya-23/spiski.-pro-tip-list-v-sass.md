# Списки

Списки в Sass відносяться до окремого типу даних для зберігання послідовності значень, які можуть бути відоремленими комами `Helvetica, Arial, sans-serif` або пробілами `10px 15px 0 0`&#x20;

Для того, щоб позначити список в Sass не потрібно використовувати ніяких спеціальних дужок. Будь-які вирази, розділені пробілами або комами, вважаються списком. Також можна використовувати круглі `(...)` або квадратні `[...]` дужки для позначення списку і кращого сприйняття коду.

```scss
// List examples
$list: 8, 10, 12, 14, 16;

$list: 1px solid red;
```

У препроцесорі Sass [вбудований модуль](https://sass-lang.com/documentation/modules/list/) `sass:list` містить набір функцій, що дають змогу використовувати списки для написання потужних бібліотек стилів або зробити таблицю стилів вашої програми чистішою та зручнішою для обслуговування.

Більшість цих функцій приймають або повертають числа, які називаються _**індексами**_, які посилаються на елементи в списку. Індекс **1** вказує на перший елемент списку. Sass також дозволяє легко посилатися на кінець списку. Індекс **-1** відноситься до останнього елемента списку, **-2** відноситься до передостаннього і так далі.

{% hint style="info" %}
Зауважте, що звернення до першого елемента списку відбувається за індексом 1 — це відрізняється від багатьох мов програмування, де індекси починаються з 0
{% endhint %}

Списки не дуже корисні, якщо ви не можете отримати з них значення. Тому для отримання значення зі списку можна використовувати функцію `list.nth($list, $n)`, щоб отримати елемент із заданим індексом у списку. Перший аргумент — це сам список, а другий — індекс значення, яке потрібно отримати.

```scss
@use 'sass:list';

$numbers: 10px, 12px, 16px;
@debug list.nth($numbers, 2); // 12px
```

Іноді буває необхідно дізнатись скільки елементів міститься у списку, тобто його довжину. Для цього можна скористатись функцією `list.length($list)`, яка поверне число — кількість елементів. Аргумент, який приймає функція — це сам список.

```scss
@use 'sass:list';

$colors: lightgreen, tomato, lightblue;
@debug list.length($colors); // 3
```

Для того, щоб пройтись по списку і згенерувати стилі для кожного елемента зі списку використовують директиви `@each` або `@for`

```scss
$sizes: 40px, 50px, 80px;

@each $size in $sizes {
  .icon-#{$size} {
    font-size: $size;
    height: $size;
    width: $size;
  }
}
```

```scss
@use 'sass:list';

$button-types: ".btn-sm", ".btn-md", ".btn-lg", ".btn-xl";

@mixin btn-size($types) {
  @for $i from 1 through list.length($types) {
    #{list.nth($types, $i)} {
      $factor: ($i * 5 - 5);

      padding: (5px + $factor) (10px + $factor);
      border-radius: (10px + $factor);
    }
  }
}

@include btn-size($button-types);
```

Списки в Sass імутабельні (незмінні). Це означає, що вміст значення списку ніколи не змінюється. Усі функції для роботи зі списками в Sass повертають нові списки, а не змінюють оригінали. Незмінність допомагає уникнути багатьох прихованих помилок, які можуть виникнути, коли той самий список використовується в різних частинах таблиці стилів.
